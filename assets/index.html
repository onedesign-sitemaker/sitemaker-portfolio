Pattern D Pattern I Pattern J sitemaker

Thin 1px border ghost button with transparent background. Hover fills from left to right via sliding pseudo-element. Arrow icon shifts right on hover. Sharp 0px radius. Light-on-dark and dark-on-light variants.

border: 1px solid border-radius: 0 padding: 16px 48px transition: 0.45s cubic-bezier font: Inter / Cormorant Garamond
Preview

</a>

</a>

  border-radius: 0;
  background: transparent;

  font-family: 'Inter', sans-serif;
  font-size: 13px;
  font-weight: 400;
